How Podcast Assistant Works

📡 Automatic Ingestion

Subscribe to any podcast via RSS. The system polls feeds every 6 hours (and on startup) to detect new episodes automatically.

🎧 Transcription & Summarization

New episodes are downloaded, transcribed using Groq Whisper, then summarized by Llama 3.3 70B. Each episode gets a structured summary, key topics, highlights, and a listen-worthiness score (1–10).

📊 Daily Digest

The Digest page shows episodes from the last 24 hours ranked by score. High scores (7+) mean "worth your time", low scores (1–3) mean "safe to skip". Hit Refresh to manually pull new episodes.

💬 Chat Agent

The Chat page lets you talk to an AI that knows your podcast library. Ask it to search episodes, summarize topics, compare shows, or get personalized recommendations. Set an objective (e.g. "I'm interested in AI") to get relevance scores alongside listen scores.

➕ Managing Podcasts

Go to Podcasts to add or remove subscriptions. Paste any RSS feed URL — episodes start processing in the background immediately.
